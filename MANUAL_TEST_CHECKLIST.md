# CardUbi 项目升级 - 手动测试清单

## 测试环境准备
- [ ] 在浏览器中打开 index.html (建议使用 Chrome/Edge)
- [ ] 确保网络连接正常（CSV 数据获取需要网络）
- [ ] 打开开发者工具控制台查看日志

## 阶段1: Insider Vault 测试

### 1.1 视觉检查
- [ ] 紧急区域是否已转换为深色玻璃态设计？
- [ ] 标题是否显示 "Insider Vault" 和 "🔒" 图标？
- [ ] 副标题是否显示 "今日精選：4 家銀行 App 內限定隱藏優惠"？
- [ ] 稀缺性说明是否显示 "由人工登入銀行 App 蒐集（外部網站搜尋不到）"？

### 1.2 数据获取和缓存
- [ ] 页面加载时是否显示 "今日精選" 而不是倒计时？
- [ ] 控制台是否显示 "獲取 CSV 數據..." 或 "使用緩存的 Insider Vault 選擇"？
- [ ] 检查 localStorage 是否有 `cardubi_vault_YYYY-MM-DD` 键？
- [ ] 刷新页面后，相同的优惠选择是否保持不变？（每日稳定性）

### 1.3 卡片交互测试
- [ ] 卡片初始状态是否为模糊/锁定状态？
- [ ] 卡片上是否显示 "🔒 點擊解鎖" 提示？
- [ ] 卡片上是否有 "全網獨家" 标签（带微妙脉冲效果）？

#### 点击测试：
- [ ] 点击卡片 - 模糊效果是否消失？
- [ ] 解锁后是否显示银行、App名称、优惠标题、结束日期、隐藏备注？
- [ ] 解锁后是否有 "重新鎖定" 按钮？
- [ ] 点击 "重新鎖定" 按钮 - 卡片是否重新变模糊？

#### 键盘可访问性测试：
- [ ] 使用 Tab 键导航到卡片
- [ ] 按 Enter 键 - 卡片是否解锁？
- [ ] 按 Space 键 - 卡片是否解锁？
- [ ] 解锁后，Tab 键是否能导航到 "重新鎖定" 按钮？
- [ ] 按 Enter/Space 在 "重新鎖定" 按钮上 - 卡片是否重新锁定？

### 1.4 边缘情况测试
- [ ] 如果 CSV 获取失败，是否显示错误信息和重试按钮？
- [ ] 如果有效银行少于4家，是否显示 "本週精選不足 4 家，持續補貨中"？
- [ ] 点击重试按钮是否重新尝试获取数据？

## 阶段2: Apple风格现金回馈计算器测试

### 2.1 视觉检查
- [ ] 计算器区域是否显示 "現金回饋計算器" 标题？
- [ ] 是否有 "Smart Rate 3.5%" 标签？
- [ ] 滑块是否为 iOS 控制中心风格（厚轨道、柔和阴影）？

### 2.2 滑块交互测试
- [ ] 拖动滑块 - 金额显示是否实时更新？
- [ ] 滑块值变化时，现金回馈计算是否实时更新？
- [ ] 现金回馈数字是否有平滑计数动画？

#### 键盘导航测试：
- [ ] 使用 Tab 键导航到滑块
- [ ] 按 → 或 ↑ 键 - 金额是否增加 1,000 TWD？
- [ ] 按 ← 或 ↓ 键 - 金额是否减少 1,000 TWD？
- [ ] 按 Home 键 - 是否跳转到 0 TWD？
- [ ] 按 End 键 - 是否跳转到 100,000 TWD？

### 2.3 计算准确性测试
- [ ] 输入 50,000 TWD - 现金回馈是否显示 1,750 TWD？ (50,000 * 3.5%)
- [ ] 输入 100,000 TWD - 现金回馈是否显示 3,500 TWD？
- [ ] 输入 0 TWD - 现金回馈是否显示 0 TWD？

### 2.4 响应式测试
- [ ] 在移动设备上，滑块是否仍然可用？
- [ ] 触摸拖动滑块是否流畅？

## 阶段3: AI每周内容网格测试

### 3.1 视觉检查
- [ ] 内容中心区域是否显示 "AI 每週內容精選"？
- [ ] 是否有 "最新趨勢" 标签？
- [ ] 是否显示6张 Apple News 风格卡片？

### 3.2 网格布局测试
- [ ] 在桌面端（宽度≥1024px）- 是否显示3列？
- [ ] 在平板端（宽度≥768px）- 是否显示2列？
- [ ] 在移动端（宽度<768px）- 是否显示1列？
- [ ] 调整浏览器大小时，网格列数是否自动调整？

### 3.3 卡片内容测试
- [ ] 每张卡片是否包含：
  - [ ] 彩色图标区域
  - [ ] 文章类别标签
  - [ ] 阅读时间
  - [ ] 文章标题
  - [ ] 文章摘要
  - [ ] 发布日期
  - [ ] "閱讀全文" 按钮

### 3.4 交互测试
- [ ] 点击 "閱讀全文" 按钮 - 是否显示提示信息？
- [ ] 使用 Tab 键导航到卡片
- [ ] 按 Enter 键 - 是否触发 "閱讀全文"？
- [ ] 滚动时，卡片是否有淡入动画？

### 3.5 滚动动画测试
- [ ] 向下滚动时，新进入视口的卡片是否有淡入效果？
